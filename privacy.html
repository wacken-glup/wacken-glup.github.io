Privacy policy

This is the privacy policy for the public instance of the Wacken Group Line Up Planner.
TLDR: The app uses Google Firebase to authenticate users and store data. All data is deletable by the user.

1. Collected data

This app can collect following types of data:

  • Username
  • Mail address
  • Space member name
  • Space member color
  • Space affiliation
  • Liked and suggested acts

    • 2. Data location

    Authentication and app data is stored by Google Firebase, a service provided by Google Ireland Ltd, Gordon House, Barrow Street, Dublin 4, Ireland. The database is located on european servers.
    Learn more about how Firebase processes your data at https://firebase.google.com/support/privacy.

    3. Data deletion

    All user data is deletable with a press on a button. Head to /settings to delete your data.

    4. Data lifetime

    User data will be, unless deleted prior, stored until after the end of the festival.

    4. External content

    This app embeds some external content.

    4.1. Spotify widget

    This app includes a spotify widget, provided by Spotify AB, Birger Jarlsgatan 61, 113 56 Stockholm, Sweden.
    The spotify widget is included for users to get a first impression of a chosen artist.
    Learn more about how spotify processes your data at https://www.spotify.com/de/legal/privacy-policy.

    4.2. Wacken media

    This app loads images from https://www.wacken.com provided by WOA Festival GmbH, Schenefelder Str. 17, 25596 Wacken, Germany.
    The images from wacken.com are included for users to get a first impression of a chosen artist.
    Learn more about how wacken.com processes your data at https://www.wacken.com/de/rechtliches/impressum/#c774.

    5. Cookies

    This app does save data in a way comparable to Cookies. This does only occur after a user has signed in or has created an account.

    6. Hosting

    This app is hosted with Github pages. The privacy policy of github does apply.
    Learn more about data collection on github.io sites at https://docs.github.com/en/pages/getting-started-with-github-pages/about-github-pages#data-collection and https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement.

    6. Open Source software

    This app is open source software. You can retrace how data is collected at https://github.com/wacken-glup/wacken-glup.
